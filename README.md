## Eclipse命令行参数使用

Run---->Run Configurations----->右边 Arguments 里 Program arguments 中写。

如果写的是文件，默认该文件要放在项目的根目录里。当然也可以换位置，勾选 Working directory 的 Other，然后选择文件目录即可。
